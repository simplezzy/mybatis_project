# mybatis_project
1.GitHub Basement:
  a. 本地项目上传到远程仓库
     i.建立本地git仓库
       git init
       git add .
       git commit -am "comment"
     ii.GitHub 建立远程仓库
        https://github.com/simplezzy/project_name.git
     iii.本地仓库提交到远程仓库
         增加远程仓库 
           git remote add origin https://github.com/simplezzy/project_name.git
           git remote -v
           git pull origin master 
           git push origin master
